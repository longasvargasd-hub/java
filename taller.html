Ejercicios en Consola - Java

Volver a la página principal

Descripción 1: En este ejercicio practiqué cómo leer datos desde la consola. El programa me pidió que ingresara algunos valores y luego los mostró en pantalla. Con esto entendí mejor cómo funciona la entrada y salida de datos en Java.
Descripción 2: Aquí hice un programa que suma o realiza una operación básica con los números que digito. Probé varios valores para asegurarme de que los resultados fueran correctos y que el programa respondiera sin errores.
Descripción 3: En este caso trabajé con condicionales. El programa me pedía un número y mostraba si era positivo, negativo o cero. Con esto comprobé cómo usar el if y el else para tomar decisiones dentro del código .

Descripción 4: Este ejercicio trataba sobre ciclos. Hice que el programa repitiera una acción varias veces usando un for. Me sirvió para entender cómo recorrer una secuencia de números y mostrar los resultados uno por uno en la consola .

Descripción 5: Aquí practiqué con un ciclo while. El programa seguía ejecutándose mientras se cumplía una condición. Me ayudó a ver cómo controlar las repeticiones sin necesidad de saber cuántas veces se va a repetir al principio.

Descripción 6: En este ejercicio hice un pequeño menú en consola. El usuario podía escoger una opción y el programa respondía según la elección. Fue útil para combinar condicionales con estructuras repetitivas.
Descripción 7: Trabajé con arreglos. El programa me pidió varios números y luego los mostró organizados o hizo alguna operación con ellos. Con esto entendí mejor cómo guardar varios valores en una sola variable y recorrerlos .

Descripción 8: Aquí el ejercicio consistía en calcular promedios. Ingresé varias notas y el programa mostró el resultado final. Fue una buena forma de practicar operaciones con variables y estructuras repetitivas.

Descripción 9: En este punto probé cómo usar funciones o métodos. Hice que una parte del código realizara un cálculo y devolviera el resultado. Me ayudó a entender cómo dividir el programa en partes más ordenadas y fáciles de leer.

Descripción 10: Este ejercicio trataba sobre el manejo de texto. Ingresé cadenas y el programa mostró algunos cambios, como mayúsculas o conteo de caracteres. Me sirvió para conocer mejor las funciones que se pueden usar con strings.

Descripción 11: Por último, hice un ejercicio más completo donde el programa unía varios conceptos: lectura de datos, ciclos y condicionales. Con este pude repasar todo lo aprendido y ver cómo aplicar varias cosas en un solo código.
